# 上海高校气质测试 v4.1 - 矩阵工作稿

## 本轮修正重点

### 复旦（重写）
- 不再按“理解强 / 逻辑强 / 会想通”去写。
- 更偏：读书人气、文化底子、非纯功利、人文关怀、自由但不急着兑现、谈吐和判断来自阅读与见识。
- 复旦像的是：身上有书卷气、不是只会做题、不是只拿结论、不是只看效率的人。
- 所以以后：
  - 降低复旦在“知识点打通 / 吃透 / 补漏洞 / 自己钻通”类题里的默认主吃。
  - 提高复旦在“真知道 / 非机械刷题 / 见识与底子 / 被知识和阅读养厚”类题里的主吃。

### 交大（补充）
- 不再只写成高压、硬度、往上顶、会卷。
- 保留：头部平台、行业龙头、核心位置、复杂系统、硬本事。
- 额外补：年轻工科导师 / 年轻老板气。
- 也就是说交大不必总是很死板、很严肃，它还应该带一点：
  - 做事有劲
  - 技术乐观
  - 有带队感
  - 能把复杂东西弄起来
  - 有工程型魅力，不只是压强和忍耐

- 不要总像“卷王 / 高压 / 苦练机器”。

- 工程型魅力
- 年轻工科导师 / 年轻老板感
- 既强又能带人，不只是自己扛
- 有项目感，但不一定很土很木

### 交大适合继续吃的题
