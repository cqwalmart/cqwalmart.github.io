====================
当前方法论（重要）
====================

一、关于学校理解的补充修正
- 不要把复旦简单理解成“逻辑链通、知识结构搭得起来”的学校。
- 复旦更该理解成一种文化高地感：人文底子、见识、阅读、谈吐、判断、文化资本、自我要求和精神气质的综合体。
- 也就是说，复旦不只是“想明白一个知识点”，而是更像“人身上有东西，有文化感，有底子，有自己判断世界的方式”。
- 所以后面配分时，凡是更偏：文化见识、谈吐、底子、真知道、不是只有做题能力、不是纯工具理性，那些入口都应该多考虑复旦。
- 反过来，纯“知识点串起来了 / 逻辑上想通了”不一定就要高给复旦，有时更该给上科、华师，或者分散给别的学校。
- 也不要把交大简单理解成“nerd / 工科脑 / 只会埋头干活”。
- 交大还应该带一点行业龙头感：强势平台、头部位置、资源密度、往上够、以后能进核心位置或核心行业的感觉。
- 所以后面配分时，凡是更偏：高标准竞争、头部平台、核心位置、行业中坚、把能力换成更高站位，这些入口都应该多考虑交大。
- 反过来，纯“会做题 / 理工 nerd / 只会练”不应该自动高给交大，要和同济、华理、上财甚至别的学校分开。

一、改题方法
1. 偏生活，不要悬
- 题目和选项要像高中生真会碰到、真会在脑子里冒出来的话。
- 少用“进到自己身上 / 活成模板 / 被收编 / 去处感 / 长牢”这种讨论稿词。
- 优先写成：作业怎么做、碰到难题怎么办、考试失手后会怎样、学校讲不明白会怎么补。

2. 抓核心，不要绕
- 选项要尽量短，先把最核心的矛盾说出来。
- 不要一段话里塞两三层意思。
- 如果一句话能说清，就不要硬写成文案腔。

3. 要有区分度，别让人每个都想选
- 每个选项都要像一种不同的处理方式，而不是四种都能共情的伤感总结。
- 用户看完最好是“别的也有一点，但我最像这个”，而不是“四个都像我”。
- 能写成具体动作、具体习惯、具体选择，就不要只写抽象态度。

4. 不要老问同一层东西
- 如果已经有题在问“你最近卡在哪”，后面的题就别再换个壳重复问一遍。
- 同一组题之间要区分：当前痛点、学习触发器、作业处理方式、补强方式、考试失手后的反应、未来想要带走什么。

二、赋分方法
1. 配分不能只按人格像谁，要按高中表现最后更容易流向哪
- 不是“这个选项像哪所学校气质”，而是“这种学习行为 / 做事方式 / 高考表现，最后更容易把人推到哪些学校”。

2. 要遍历 12 所学校
- 每个选项都不能只抓两三所最贴的学校。
- 要一个学校一个学校想：这种人会不会流向它？它常见学生和这个行为像不像？
- 主命中 / 次命中 / 边缘入口都可以有。

3. 先立逻辑，再慢慢磨小数
- 现阶段先把方向想对：高分型、稳分型、策略型、状态型、探索型、执行型分别更容易流向哪。
- 细分分值后面再慢慢调。
