# 上海高校气质测试 v4 - 矩阵工作稿

## 工作方法

### Step 1：按题修矩阵
对每道题的四个选项分别检查：
- 主命中学校是否合理
- 次命中 / 边缘入口是否需要补
- 是否误把学校当成人格标签在喂分
- 是否和已修正的学校理解冲突（尤其复旦、交大）

### Step 2：按学校回看整套题
对 12 所学校逐一检查：
- 这套 16 题里入口够不够
- 哪些入口过厚 / 过薄
- 是否被错误题型喂太多
- 有没有关键气质完全没入口

## 当前学校理解修正（本轮必须遵守）
- 复旦：更偏文化高地、底子、阅读、谈吐、判断、真知道；不能简单吃“知识点想通了”。
- 交大：不只是 nerd / 会练；还带头部平台、行业龙头、核心位置、往上够的感觉。
- 上科：非标准、探索、创造力、实验性、往未知里长。
- 同济：体系训练、工程门派、项目磨出来。
- 华理：靠谱、耐干、顶用、现实硬度。
- 华政：站位、判断、分寸、说得住。
- 上财：路径、落点、收益感、早点把路想清楚。
- 华师：顺、人味、理解顺、别把人过坏。
- 东华：状态、样子、质感、生活方式。
- 上外：交流质量、语境、互动感。
- 上大：不最标准，但混着长、自己拎、现实韧性。
- 上纽：另一套人生排法、另一种生活节奏。

## v4-1 你最近最常卡住你的，更像是哪一种

### A 学了留不住
暂定：复旦 / 华师 / 上科 / 少量上大
原因：
- 不是单纯不会，是“学了但没真进去”。
- 复旦吃“真进去没进去”；华师吃“理解顺不顺”；上科少量吃探索型吸收；上大少量吃现实中的“混着学、混着长”。

### B 赶进度只能应付
暂定：交大 / 华理 / 同济 / 少量上财
原因：
- 是节奏和硬度不够，不是单纯方向问题。
- 交大吃高压和往上顶；华理吃硬度和顶用；同济吃训练感；上财少量吃“效率不足”的焦虑。

### C 一直吊着一口气
暂定：华师 / 东华 / 上大 / 少量上外
原因：
- 这是状态和生活方式问题。
- 华师吃“别把人过坏”；东华吃状态和质感；上大吃现实中不太顺但还得过；上外少量吃互动和氛围敏感。

### D 为什么一定要按这套来
暂定：上科 / 上纽 / 上大 / 少量华政
原因：
- 不是路径迷茫，而是对标准轨道起疑。
- 上科吃非标准；上纽吃另一套人生排法；上大吃不最标准但自己长；华政少量吃“先判断这套值不值”。
